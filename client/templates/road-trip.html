Road Trip

Road Trip was designed to create a detailed itinerary for any road trip. This will include all activities, resturaunts, hotels, and any other here and now places you would like to go. The user can add legs to the trip, choose where they want to stop, and plan ahead to make sure their trip is the best experience possible. My final project in the class using resources outside of curriculum. React is awesome and it was nice exposure to learning frameworks on my own from documentation. Working with several parts within Google Maps API was interesting as well. My favorite project I worked on, with many more features to come as I dive deeper into the project.
